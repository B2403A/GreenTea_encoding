绿茶一键压制脚本


程序旨在实现一键压制字幕视频，包括软压MKV和硬压MP4

程序仅限交流学习，不可用于商用，一经发现将全网公示警告

*不要用这个脚本通过绿茶的测试，你会后悔的*


使用方法：

*本程序支持windows、linux和mac，请自行部署python环境*

1.将程序包解压到根目录，尽量不要包含特殊字符以及中文路径；

2.运行requirement.py文件；

3.将源视频放在/raw，字体放在/fonts，字幕放在/ass；

4.运行autochapter.py，将半自动生成章节文件，请注意一定要人工检查XML和ffmeta时间戳！

5.运行launch.py；若使用终端请确保在脚本所在根目录下运行，然后执行:python3 launch.py；

6.按照提示输入番剧的英文名；

7.选择硬压（MP4）或软压（MKV）；

8.输出在/out文件夹
